 # E-commerce Linked List Sorting

## Overview
This project demonstrates sorting algorithms applied to linked lists in an e-commerce context.

The goal is to efficiently organise product data using structured data handling and algorithmic logic.

## Features
- Linked list implementation
- Sorting algorithms applied to product data
- Structured data processing
- Algorithm efficiency considerations

## Technologies

- Jupyter Notebook

## Relevance
This project demonstrates problem-solving, algorithmic thinking and structured logic, relevant for software development and testing.
